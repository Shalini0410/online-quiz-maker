# Project Walkthrough - Online Quiz Maker 🧠

The **Online Quiz Maker** is a premium web platform that enables users to create and take interactive quizzes. It features a modern, responsive design with glassmorphism aesthetics and smooth animations.

## ✨ Accomplishments

- ✅ **Full User Authentication**: Registration and Login flows implemented.
- ✅ **Dynamic Quiz Creation**: A multi-step builder for creating custom quizzes with multiple-choice questions.
- ✅ **Interactive Taking Interface**: A progress-tracked interface for answering questions one-by-one.
- ✅ **Instant Scoring & Review**: Immediate calculation of results with a detailed review of correct vs. chosen answers.
- ✅ **Mobile-First Design**: Fully responsive layout optimized for all screen sizes.
- ✅ **Premium UX**: Smooth transitions using Framer Motion and elegant icons from Lucide.

## 📺 Demo Highlights

### 1. Home Page & Landing

## 🛠️ Deployment Instructions

Since the browser environment had technical difficulties, I've prepared everything for you to finish the push to GitHub:

### 1. Create Repository
Go to [GitHub New Repo](https://github.com/new) and create a repository named `online-quiz-maker`.

### 2. Push Code
Open your terminal in the project directory and run:
```bash
git remote add origin https://github.com/Shalini0410/online-quiz-maker.git
git branch -M main
git push -u origin main
```

### 3. Live Link
For a live preview, you can deploy to **GitHub Pages** (Settings > Pages > Build from main branch) or use **Vercel** (connect your repository).
